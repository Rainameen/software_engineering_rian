QuerySet Operations Reference
Basic Operations
  • order_by('field') - Sort ascending
  • order_by('-field') - Sort descending
  • filter(field=value) - Include matching records
  • exclude(field=value) - Exclude matching records
Field Lookups
  • field__contains='text' - Contains substring
  • field__in=[val1, val2] - In list of values
  • field__range=[start, end] - Within range
  • field__exact=value - Exact match
Pro Tip

These QuerySet operations can be chained together for more complex queries. For example: Student.objects.filter(is_active=True).order_by('-gpa').exclude(year='4')
